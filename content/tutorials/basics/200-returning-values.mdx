---
title: 値の返却
excerpt: Effectの基本を学ぶ

最初のEffectを実行できておめでとうございます！

次に、Effectから値を返す様々な方法を見ていきましょう。

### Effect.succeed

毎回同じ値を返すエフェクトを作成するには、`Effect.succeed`コンストラクタを使用できます。これは値を入力として受け取り、その値で成功するエフェクトを生成します。以下は例です：

```ts
Effect.succeed(42)
```

このエフェクトは常に値`42`で成功します。

### Effect.sync

各実行ごとに同期関数を実行するエフェクトを作成したい場合は、`Effect.sync`コンストラクタを使用できます。これは引数のない関数を入力として受け取り、各実行ごとにその関数を呼び出します。以下は例です：

```ts
Effect.sync(() => Date.now())
```

このエフェクトは、実行されるたびに現在の日付を返します。もし代わりに`Effect.succeed(Date.now())`を使用した場合、日付はエフェクトが作成された時点で固定されてしまうことに注意してください。

### Effect.promise

Promiseを返す非同期関数からエフェクトを作成したい場合は、`Effect.promise`コンストラクタを使用できます。結果として得られるエフェクトは、返されたPromiseを解決し、その値で成功します。以下は例です：

```ts
Effect.promise(() => fetch('https://api.github.com/users/octocat'))
```

このエフェクトはGitHubユーザー `octocat` を取得し、そのレスポンスで成功します。
