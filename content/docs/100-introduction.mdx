Effectのドキュメントへようこそ！

## Effectとは何ですか？

Effectは、開発者が複雑な同期および非同期プログラムを簡単に作成できるように設計された強力なTypeScriptライブラリです。

## 主な機能

Effectの主な機能には以下が含まれます：

| **機能**               | **説明**                                                                                                     |
| ---------------------- | ------------------------------------------------------------------------------------------------------------ |
| **並行性**             | Effectのファイバーベースの並行性モデルを通じて、高度にスケーラブルで超低遅延のアプリケーションを実現します。 |
| **構成可能性**         | 小さく再利用可能なビルディングブロックを使用して、高度に保守可能で読みやすく柔軟なソフトウェアを構築します。 |
| **リソースの安全性**   | プログラムが失敗した場合でも、リソースの取得と解放を安全に管理します。                                       |
| **型の安全性**         | TypeScriptの型システムを最大限に活用し、Effectの型推論と型の安全性に焦点を当てます。                         |
| **エラーハンドリング** | Effectの組み込みのエラーハンドリング機能を使用して、エラーを構造化された信頼性のある方法で処理します。       |
| **非同期性**           | 同期的であれ非同期的であれ、同じように見えるコードを書きます。                                               |
| **可観測性**           | 完全なトレース機能を備えており、Effectプログラムの実行を簡単にデバッグおよび監視できます。                   |

## ドキュメントの使い方

ドキュメントは基本から始まり、より高度なトピックへと進む順序で構成されています。これにより、Effectアプリケーションを構築する際にステップバイステップで進むことができます。ただし、ドキュメントを任意の順序で読むことも、特定のユースケースに関連するページに直接ジャンプすることも可能です。

ページ内のナビゲーションを容易にするために、画面の右側にページの目次があります。これにより、ページ内の異なるセクション間を簡単に移動することができます。
